# 自動コミットの起動方法

## PC起動後に自動で動かす（推奨）

1. **タスクを再登録**（設定を変更したため、1回だけ実行）
   - PowerShell で次を実行：
   ```powershell
   cd "c:\Users\20171\IT_Learning\pre-joining-learning\LearningTools\auto-git-commit"
   .\setup-auto-start.ps1
   ```
   - エラーが出た場合は、PowerShell を**管理者として開いて**同じコマンドを実行

2. 次回からは **ログオンして約1分後** に自動でスクリプトが起動します。

---

## 今すぐ手動で動かす

### 方法A: タスクを開始する
```powershell
Start-ScheduledTask -TaskName "Auto Git Commit"
```

### 方法B: スクリプトを直接実行する
```powershell
cd "c:\Users\20171\IT_Learning\pre-joining-learning\LearningTools\auto-git-commit"
.\auto-git-commit.ps1
```
（このウィンドウは閉さないでください。閉じると自動コミットも止まります。）

---

## ログの場所

- リポジトリルート: `pre-joining-learning\.git-auto-commit\logs\log-日付.txt`
- スクリプト側: `LearningTools\auto-git-commit\.git-auto-commit\logs\` （タスクで起動した場合にここに書かれる場合があります）

## タスクを削除する場合

```powershell
cd "c:\Users\20171\IT_Learning\pre-joining-learning\LearningTools\auto-git-commit"
.\setup-auto-start.ps1 -Remove
```
